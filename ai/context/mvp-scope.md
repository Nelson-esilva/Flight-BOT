# Escopo do MVP

## Dentro do MVP

- Criar monitor.
- Listar monitores.
- Pausar/remover monitor.
- Executar busca manual.
- Execucao periodica.
- Consultar Amadeus.
- Salvar resultados.
- Aplicar regra.
- Enviar Telegram.
- Evitar alerta duplicado.

## Fora do MVP

- WhatsApp.
- Frontend web.
- Autenticacao multiusuario.
- Scraping.
- Compra automatica.
- Multiplas fontes.
- Recomendacao com IA.
- Historico grafico.
- Previsao de preco.
- Milhas.
- Login em companhias aereas.

## Regra de decisao

Se uma tarefa nao estiver explicitamente dentro do MVP, nao implemente sem confirmacao do usuario.
